# 试验项目概览

## 🔍 如何选择试验项目？

选择试验项目时，应考虑以下因素：

1. **产品使用环境** - 根据产品实际应用场景选择对应的环境应力
2. **行业要求** - 不同行业（汽车、军工、轨道交通）有特定的测试要求
3. **客户要求** - 满足客户技术协议中规定的测试条件
4. **产品设计特点** - 根据产品材料、结构特点确定重点测试项目
