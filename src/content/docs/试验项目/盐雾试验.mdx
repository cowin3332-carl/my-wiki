# 盐雾试验

## 一、测试目的

盐雾试验用于评估电工电子产品及材料在**盐雾腐蚀环境**下的适应能力，检验金属镀层、涂层、材料本身的耐腐蚀性能。

## 二、核心标准

| 标准号 | 标准名称 |
|--------|---------|
| GB/T 2423.17 | 电工电子产品环境试验 试验 Ka：盐雾 |
| IEC 60068-2-11 | Environmental testing - Test Ka: Salt mist |
| GB/T 2423.18 | 电工电子产品环境试验 试验 Kb：交变盐雾 |
| IEC 60068-2-52 | 交变盐雾试验 |
| GB/T 10125 | 人造气氛腐蚀试验 盐雾试验 |
| ISO 9227 | 人造气氛中的腐蚀试验 - 盐雾试验 |

## 三、试验类型

### 1. 中性盐雾试验 (NSS)

- 溶液：5% NaCl，pH 6.5-7.2
- 适用于一般金属及镀层
- 最常用的盐雾试验方法

### 2. 乙酸盐雾试验 (AAS)

- 溶液：5% NaCl + 乙酸，pH 3.1-3.3
- 腐蚀速度更快
- 适用于装饰性镀铬层

### 3. 铜加速乙酸盐雾 (CASS)

- 溶液：5% NaCl + 乙酸 + 氯化铜
- 腐蚀速度最快
- 适用于铜 + 镍 + 铬多层镀层

## 四、典型试验条件

| 参数 | 条件 |
|-----|------|
| 盐溶液浓度 | 50 ± 5 g/L (5%) |
| 试验箱温度 | 35 ± 2°C |
| 饱和桶温度 | 47 ± 2°C |
| 盐雾沉降率 | 1-2 mL/80cm²·h |
| 喷雾方式 | 连续喷雾 / 交变喷雾 |

## 五、试验周期

| 产品类型 | 试验时间 |
|---------|---------|
| 一般电子产品 | 24h, 48h |
| 汽车电子 | 48h, 96h, 240h |
| 海洋设备 | 240h, 480h, 1000h |
| 军工产品 | 按规范要求 |

## 六、评级方法

### 外观评级

| 等级 | 腐蚀面积比例 | 说明 |
|-----|-------------|------|
| 10 | 0% | 无腐蚀 |
| 9 | &lt;0.1% | 轻微腐蚀 |
| 8 | 0.1-0.25% | 轻度腐蚀 |
| 7 | 0.25-0.5% | 中等腐蚀 |
| 6 | 0.5-1.0% | 较严重腐蚀 |
| ≤5 | &gt;1.0% | 严重腐蚀 |

### 保护评级 (Rp) / 外观评级 (Ra)

按 GB/T 6461 进行评级。

## 七、常见失效模式

- [ ] 金属表面生锈、腐蚀
- [ ] 镀层起泡、剥落
- [ ] 涂层粉化、脱落
- [ ] 紧固件锈死
- [ ] 接触电阻增大
- [ ] 结构强度下降

## 八、试样制备与处理

```
样品制备 → 清洁处理 → 原始检测 → 放入盐雾箱
                                        ↓
    结果评定 ← 干燥处理 ← 清洗除盐 ← 规定时间取出
```

## 九、注意事项

1. **溶液配制**：使用化学纯 NaCl 和去离子水
2. **样品放置**：与垂直方向成 15-25°角
3. **箱内清洁**：定期清除沉积盐分
4. **喷雾控制**：保持连续均匀喷雾
5. **废液处理**：按规定处理含盐废液
