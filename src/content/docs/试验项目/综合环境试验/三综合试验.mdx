# 三综合试验

## 一、测试目的

三综合试验用于评估电工电子产品在**温度、湿度、振动综合作用**下的适应能力，更真实地模拟实际使用环境，发现单一应力无法暴露的缺陷。

## 二、核心标准

| 标准号 | 标准名称 |
|--------|---------|
| GJB 150.9A | 湿热试验（带振动） |
| MIL-STD-810H | Method 520.4 Combined Environments |
| GB/T 14583 | 电工电子产品环境试验 综合试验导则 |

## 三、为什么需要三综合试验？

实际使用环境中，产品往往同时承受多种环境应力：

| 单一应力试验 | 三综合试验 |
|------------|-----------|
| 仅考核单一因素 | 多应力综合作用 |
| 可能遗漏缺陷 | 更易暴露潜在问题 |
| 简单、成本低 | 复杂、成本高 |
| 适用于初筛 | 适用于定型试验 |

### 综合应力效应

```
单一应力 A + 单一应力 B < 综合应力 (A+B)

协同效应使失效风险倍增！
```

## 四、试验类型

### 1. 温湿度 + 振动三综合

最常用的组合，模拟气候 + 机械综合环境。

### 2. 温度循环 + 振动

评估热应力与机械应力共同作用。

### 3. 恒定湿热 + 振动

评估潮湿环境下振动的腐蚀加速效应。

## 五、典型试验条件

### 条件示例

| 参数 | 条件 A | 条件 B | 条件 C |
|-----|-------|-------|-------|
| 温度范围 | -20~+60°C | -40~+70°C | -55~+85°C |
| 湿度范围 | 20~95%RH | 20~95%RH | 20~95%RH |
| 振动类型 | 随机振动 | 随机振动 | 正弦扫频 |
| 振动量级 | 0.15g²/Hz | 0.04g²/Hz | 5g 峰值 |
| 循环次数 | 3 循环 | 5 循环 | 10 循环 |

## 六、试验设备

| 设备类型 | 组成 |
|---------|------|
| 三综合试验箱 | 温湿度箱 + 振动台集成 |
| 延伸台面 | 将振动传递到箱内 |
| 控制系统 | 温湿度 + 振动协同控制 |

## 七、常见失效模式

- [ ] 焊点疲劳开裂（热应力 + 振动）
- [ ] 连接器松动（振动 + 湿热）
- [ ] 涂层起泡脱落（湿热 + 振动）
- [ ] 应力腐蚀开裂（腐蚀 + 应力）
- [ ] 材料老化加速（温度 + 湿度）
- [ ] 密封失效（综合老化）

## 八、试验程序

```
1. 初始检测
   ↓
2. 样品安装 (注意传感器布置)
   ↓
3. 设备校准 (温湿度 + 振动)
   ↓
4. 执行综合试验程序
   ↓
5. 中间检测 (可选)
   ↓
6. 最后检测与评定
```

## 九、注意事项

1. **台面选择**：延伸台面可能引入共振，需验证
2. **结露控制**：温度变化时防止样品表面结露
3. **传感器布置**：在样品关键位置布置温湿度、振动传感器
4. **控制精度**：温湿度与振动的协同控制精度
5. **安全保护**：防止水汽进入振动台
