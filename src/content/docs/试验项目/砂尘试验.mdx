# 砂尘试验

## 一、测试目的

砂尘试验用于评估电工电子产品在**风沙环境**下的适应能力，检验产品外壳防护能力以及沙尘侵入对内部元器件的影响。

## 二、核心标准

| 标准号 | 标准名称 |
|--------|---------|
| GB/T 2423.37 | 电工电子产品环境试验 试验 Lc：导尘 |
| IEC 60068-2-68 | Environmental testing - Test L: Dust and sand |
| GB/T 4208 | 外壳防护等级 (IP 代码) |
| IEC 60529 | Degrees of protection by enclosures |
| GJB 150.12A | 砂尘试验 |
| MIL-STD-810H | Method 510.6 Sand and Dust |

## 三、试验类型

### 1. 导尘试验 (Lc1)

- 试验箱内粉尘浓度较低
- 模拟室内或轻度沙尘环境
- 主要用于 IP5X 防护等级测试

### 2. 吹砂试验 (Lc2)

- 用压缩空气吹送沙尘
- 模拟风沙环境
- 风速可达 1.5-30 m/s

### 3. 吹尘试验 (Lc3)

- 用压缩空气吹送细尘
- 模拟扬尘环境
- 主要用于 IP6X 防护等级测试

## 四、试验材料

### 沙尘成分

| 成分 | 二氧化硅 | 氧化铝 | 氧化铁 | 其他 |
|-----|---------|-------|-------|------|
| 典型含量 | 70-80% | 10-15% | 3-5% | &lt;5% |

### 粒径要求

| 试验类型 | 粒径范围 |
|---------|---------|
| 滑石粉 (IP 测试) | ≤75μm |
| 吹砂试验 | 149-841μm |
| 吹尘试验 | ≤150μm |

## 五、典型试验条件

| 参数 | 条件 |
|-----|------|
| 粉尘浓度 | 0.1-1.0 kg/m³ |
| 试验温度 | 15-35°C |
| 风速 | 1.5-30 m/s (吹砂) |
| 持续时间 | 2-8h (IP 测试) / 更长 (环境模拟) |
| 真空度 | ≤2kPa (IP6X 测试) |

## 六、IP 防护等级测试

### 防尘等级

| IP 代码 | 防护等级 | 测试条件 |
|--------|---------|---------|
| IP5X | 防尘 | 滑石粉，≤2kg/m³，2-8h，允许少量进入 |
| IP6X | 尘密 | 滑石粉，≤2kg/m³，8h，真空 2kPa，不允许进入 |

## 七、常见失效模式

- [ ] 沙尘侵入导致电路短路
- [ ] 活动部件磨损、卡滞
- [ ] 接插件接触不良
- [ ] 散热孔堵塞
- [ ] 光学部件污染
- [ ] 密封失效

## 八、试验程序

```
1. 初始检测 → 2. 样品安装 → 3. 设置参数
                              ↓
    6. 最后检测 ← 5. 清灰处理 ← 4. 执行试验
```

## 九、注意事项

1. **粉尘处理**：试验后应妥善处理粉尘，避免污染
2. **设备清洁**：定期清理试验箱内部
3. **样品处理**：试验后先清除表面粉尘再检测
4. **安全防护**：操作人员应佩戴口罩等防护用品
5. **真空条件**：IP6X 测试需抽真空模拟热胀冷缩
