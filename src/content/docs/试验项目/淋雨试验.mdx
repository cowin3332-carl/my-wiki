# 淋雨试验

## 一、测试目的

淋雨试验用于评估电工电子产品在**淋雨、滴水、喷水、浸水**等水环境下的适应能力，检验产品外壳防水性能及内部防护能力。

## 二、核心标准

| 标准号 | 标准名称 |
|--------|---------|
| GB/T 2423.38 | 电工电子产品环境试验 试验 R：水 |
| IEC 60068-2-18 | Environmental testing - Test R: Water |
| GB/T 4208 | 外壳防护等级 (IP 代码) |
| IEC 60529 | Degrees of protection by enclosures |
| GJB 150.8A | 淋雨试验 |
| MIL-STD-810H | Method 506.6 Rain |

## 三、试验类型

### IP 防水等级测试

| IP 代码 | 试验类型 | 试验条件 |
|--------|---------|---------|
| IPX1 | 垂直滴水 | 1mm/min，10min |
| IPX2 | 倾斜滴水 | 倾斜 15°，1mm/min，10min |
| IPX3 | 淋水试验 | 摆管±60°，0.7L/min，10min |
| IPX4 | 溅水试验 | 摆管±180°，0.7L/min，10min |
| IPX5 | 喷水试验 | 喷嘴 3m，12.5L/min，3min |
| IPX6 | 强烈喷水 | 喷嘴 3m，100L/min，3min |
| IPX7 | 短时浸水 | 1m 水深，30min |
| IPX8 | 持续浸水 | 按规范，>1m 水深 |
| IPX9K | 高温高压喷水 | 80°C，8-10MPa，近距离 |

### 环境淋雨测试

- **循环淋雨**：模拟自然降雨环境
- **角式淋雨**：模拟风雨交加环境
- **强化淋雨**：加速防水性能评估

## 六、常见失效模式

- [ ] 水侵入导致电路短路
- [ ] 金属部件腐蚀
- [ ] 接插件绝缘下降
- [ ] 显示屏进水
- [ ] 密封老化失效
- [ ] 呼吸阀失效

## 七、评定方法

试验后检查：

1. **外观检查**：观察是否有水迹侵入
2. **功能测试**：通电运行检查功能是否正常
3. **绝缘测试**：测量绝缘电阻、耐压强度
4. **干燥处理**：必要时干燥后复测

## 八、注意事项

1. **水温**：试验用水温度应接近样品温度，温差≤5°C
2. **水质**：使用清洁水，避免堵塞喷嘴
3. **样品状态**：通常在不工作状态下测试，特殊要求除外
4. **恢复处理**：试验后应适当干燥再检测
5. **安全措施**：电器设备浸水测试后需充分干燥
