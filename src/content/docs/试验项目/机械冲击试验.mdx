# 机械冲击试验

## 一、测试目的

机械冲击试验用于评估电工电子产品在**机械冲击环境**下的适应性，检验产品在搬运、安装、使用过程中遭受冲击时的结构强度和工作可靠性。

## 二、核心标准

| 标准号 | 标准名称 |
|--------|---------|
| GB/T 2423.5 | 电工电子产品环境试验 试验 Ea：冲击 |
| IEC 60068-2-27 | Environmental testing - Test Ea: Shock |
| GJB 150.18A | 冲击试验 |
| MIL-STD-810H | Method 516.7 Shock |

## 三、冲击波形

### 1. 半正弦波

最常用的冲击波形，模拟弹性碰撞。

```
加速度 (g)
  │
  │     /\
  │    /  \
  │   /    \
  │  /      \
  │ /        \
  │/          \
  └─────────────── 时间 (ms)
       D 持续时间
```

### 2. 后峰锯齿波

模拟刚性碰撞，冲击响应更严酷。

### 3. 梯形波

模拟爆炸冲击等特殊场景。

## 四、典型试验条件

| 应用场景 | 峰值加速度 | 持续时间 | 波形 |
|---------|-----------|---------|------|
| 一般电子产品 | 30g | 11ms | 半正弦 |
| 工业设备 | 50g | 11ms | 半正弦 |
| 汽车电子 | 100g | 6ms | 半正弦 |
| 军工产品 | 100-500g | 1-3ms | 半正弦/梯形 |
| 运输包装 | 50-150g | 6-11ms | 半正弦 |

## 五、冲击方向

一般进行**六个方向**的冲击：

- X 向：+X, -X
- Y 向：+Y, -Y
- Z 向：+Z, -Z

每轴向冲击 3 次，共 18 次。

## 六、试验设备

| 设备类型 | 特点 | 适用范围 |
|---------|------|---------|
| 冲击台 | 气动/液压驱动 | 常规冲击试验 |
| 碰撞台 | 机械式 | 小件产品 |
| 摆锤式 | 简单可靠 | 大能量冲击 |
| 落台式 | 自由落体 | 包装件测试 |

## 七、常见失效模式

- [ ] 元器件脱落、移位
- [ ] 焊点开裂
- [ ] 结构件变形、断裂
- [ ] 紧固件松动
- [ ] 接插件损坏
- [ ] 显示屏破裂
- [ ] 电池损坏
- [ ] 功能异常

## 八、试验程序

```
1. 初始检测 → 2. 安装样品 → 3. 设置参数
                              ↓
    6. 最后检测 ← 5. 六个方向冲击 ← 4. 预试验检查
```

## 九、注意事项

1. **安装方式**：模拟实际使用状态安装
2. **冲击响应**：监测关键位置的冲击响应
3. **波形选择**：根据实际环境选择合适的波形
4. **脉冲容差**：峰值加速度±15%，持续时间±10%
