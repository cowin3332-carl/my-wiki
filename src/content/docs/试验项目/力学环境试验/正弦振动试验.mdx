# 正弦振动试验

## 一、测试目的

正弦振动试验用于评估电工电子产品在**正弦振动环境**下的适应性，检验产品结构强度和元器件安装的牢固性，发现共振点并评估其影响。

## 二、核心标准

| 标准号 | 标准名称 |
|--------|---------|
| GB/T 2423.10 | 电工电子产品环境试验 第 2 部分：试验 Fc：振动（正弦） |
| IEC 60068-2-6 | Environmental testing - Test Fc: Vibration (sine wave) |
| GJB 150.16A | 振动试验 |
| GB/T 21563 | 轨道交通 冲击和振动试验 |

## 三、试验方法

### 1. 扫频振动

在规定频率范围内以一定速率往复扫频，用于：
- 寻找产品共振点
- 评估共振时的响应
- 结构性强度筛选

### 2. 定频振动

在特定频率下进行长时间振动，用于：
- 模拟特定转速设备的振动
- 共振点耐久性考核
- 运输振动模拟

## 四、试验参数

### 扫频振动参数示例

| 参数 | 条件示例 |
|-----|---------|
| 频率范围 | 10-500 Hz |
| 扫频速率 | 1 oct/min |
| 振幅/加速度 | 0.35mm / 5g (交越频率 57Hz) |
| 扫频方向 | X/Y/Z三轴向 |
| 循环次数 | 每轴向 5 次循环 |

### 定频振动参数示例

| 参数 | 条件示例 |
|-----|---------|
| 频率 | 共振点频率 或 指定频率 |
| 振幅/加速度 | 按产品规范 |
| 持续时间 | 30min / 轴向 |
| 振动方向 | X/Y/Z三轴向 |

## 六、试验设备

| 设备类型 | 特点 | 适用场景 |
|---------|------|---------|
| 电动振动台 | 频率范围宽、控制精确 | 实验室精密测试 |
| 电磁振动台 | 结构简单、成本低 | 一般筛选试验 |
| 机械振动台 | 低频性能好 | 大件产品测试 |

## 七、常见失效模式

- [ ] 紧固件松动、脱落
- [ ] 元器件引线疲劳断裂
- [ ] 焊点开裂
- [ ] 结构件变形、开裂
- [ ] 接插件接触不良
- [ ] 线圈松散
- [ ] 紧固件应力断裂

## 八、试验程序

```
1. 初始检测 → 2. 安装样品 → 3. 预检传感器
                          ↓
    5. 最后检测 ← 4. 执行振动试验
```

## 九、注意事项

1. **夹具设计**：确保夹具刚度足够，避免引入额外共振
2. **传感器布置**：在样品关键位置布置加速度传感器
3. **方向选择**：一般进行 X/Y/Z 三个轴向的测试
4. **共振搜索**：先进行低量级扫频寻找共振点
