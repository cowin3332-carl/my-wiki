# 随机振动试验

## 一、测试目的

随机振动试验用于评估产品在**随机振动环境**下的适应性，更真实地模拟实际运输、使用环境中的振动情况，考核产品结构强度和疲劳寿命。

## 二、核心标准

| 标准号 | 标准名称 |
|--------|---------|
| GB/T 2423.56 | 电工电子产品环境试验 试验 Fh：随机振动 |
| IEC 60068-2-64 | Environmental testing - Test Fh: Vibration, random |
| GJB 150.16A | 振动试验 |
| MIL-STD-810H | Method 514.7 Vibration |

## 三、随机振动特点

与正弦振动相比，随机振动：

| 特点 | 说明 |
|-----|------|
| 频谱宽 | 同时激励多个频率成分 |
| 不重复 | 瞬时值不可预测 |
| 更真实 | 更接近实际环境 |
| 统计性 | 用统计参数（PSD、RMS）描述 |

## 四、关键参数

### 功率谱密度 (PSD)

描述振动能量在频率域的分布：

```
PSD (g²/Hz)
  │
  │     ┌──────────┐
  │     │          │
  │     │          │
  │    ╱│          │╲
  │   ╱ │          │ ╲
  │  ╱  │          │  ╲
  │ ╱   │          │   ╲
  │╱    │          │    ╲
  └─────┴──────────┴─────┴──── 频率 (Hz)
       f1          f2
```

### 总均方根值 (Grms)

```
Grms = √(∫ PSD df)
```

表示随机振动的总能量。

## 五、典型试验谱

### 运输环境谱（参考）

| 频率 (Hz) | PSD (g²/Hz) |
|----------|------------|
| 10-50 | +3dB/oct |
| 50-800 | 0.04 |
| 800-2000 | -3dB/oct |
| 2000-5000 | 0.01 |

**总 Grms**: 约 6.06 g

###  MIL-STD-810 标准谱

| 频率 (Hz) | PSD (g²/Hz) |
|----------|------------|
| 20-100 | +6dB/oct |
| 100-1000 | 0.20 |
| 1000-2000 | -6dB/oct |

**总 Grms**: 约 12.3 g

## 六、试验参数设置

| 参数 | 推荐设置 |
|-----|---------|
| 频率范围 | 20-2000 Hz (根据产品) |
| 试验时间 | 每轴向 1-4 小时 |
| 控制方式 | 闭环控制 |
| 振动方向 | X/Y/Z三轴向 |
| 谱型容差 | ±3dB |

## 七、常见失效模式

- [ ] 疲劳断裂（焊点、引线、紧固件）
- [ ] 紧固件松动
- [ ] 磨损（接插件、活动部件）
- [ ] 元器件脱落
- [ ] 结构件开裂
- [ ] 涂层剥落

## 八、试验程序

```
1. 初始检测
   ↓
2. 安装传感器
   ↓
3. 控制系统校准
   ↓
4. 执行振动试验 (X/Y/Z 轴向)
   ↓
5. 最后检测与评定
```

## 九、注意事项

1. **响应监测**：在样品关键位置布置响应传感器
2. **谱型匹配**：确保实际 PSD 谱与目标谱容差在±3dB 内
3. **时间压缩**：可使用 Fatigue Damage Spectrum 进行加速
4. **台体检查**：试验前确认振动台台面空载响应
