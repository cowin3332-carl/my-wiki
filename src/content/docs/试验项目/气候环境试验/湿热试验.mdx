# 湿热试验

## 一、测试目的

湿热试验用于评估产品在**高温高湿环境**下的适应能力，检验湿度和温度共同作用下产品的耐潮湿性能，以及是否会发生凝露、腐蚀、绝缘下降等问题。

## 二、核心标准

| 标准号 | 标准名称 |
|--------|---------|
| GB/T 2423.3 | 电工电子产品环境试验 试验 Cab：恒定湿热 |
| IEC 60068-2-78 | Environmental testing - Test Cab: Damp heat, steady state |
| GB/T 2423.4 | 电工电子产品环境试验 试验 Db：交变湿热 |
| IEC 60068-2-30 | Environmental testing - Test Db: Damp heat, cyclic |

## 三、试验类型

### 1. 恒定湿热试验

温度和湿度保持恒定，用于评估长期湿热环境的影响。

| 试验条件 | 温度 (°C) | 相对湿度 (%RH) |
|---------|----------|---------------|
| 标准条件 | 40 ± 2 | 93 ± 3 |
| 热带条件 | 55 ± 2 | 93 ± 3 |

### 2. 交变湿热试验

温度和湿度按周期交变变化，模拟昼夜环境变化，可产生凝露效应。

| 参数 | 条件 |
|-----|------|
| 高温阶段 | 40/55°C, 90-98%RH |
| 低温阶段 | 25°C, 自然下降 |
| 循环周期 | 24h/循环 |
| 循环次数 | 6/21/56 天 |

## 四、试验程序

```
┌────────────────────────────────────────────┐
│  初始检测：外观、电气性能、功能检测        │
│         ↓                                  │
│  预热处理：样品放入试验箱，稳定温度        │
│         ↓                                  │
│  湿热暴露：在规定温湿度下保持规定时间      │
│         ↓                                  │
│  恢复处理：标准大气条件下恢复 1-2h         │
│         ↓                                  │
│  最后检测：与初始数据对比，评定是否合格    │
└────────────────────────────────────────────┘
```

## 五、湿热对产品的典型影响

| 影响类型 | 具体表现 |
|---------|---------|
| 金属腐蚀 | 电化学腐蚀加速，焊点、端子氧化 |
| 绝缘下降 | 吸湿导致绝缘电阻降低，漏电流增大 |
| 材料变形 | 吸湿膨胀，尺寸变化 |
| 霉菌生长 | 有机材料表面发霉 |
| 凝露短路 | 温度变化时表面结露造成短路 |
| 分层剥离 | PCB、复合材料吸湿后分层 |

## 六、评定方法

试验后检查项目：

- [ ] 外观检查：锈蚀、霉斑、变形、涂层起泡
- [ ] 电气性能：绝缘电阻、耐压强度、接触电阻
- [ ] 机械性能：操作力、配合间隙
- [ ] 功能测试：通电运行是否正常

## 七、注意事项

1. **水质要求**：加湿用水应为去离子水或蒸馏水
2. **样品放置**：避免水滴直接滴落到样品上
3. **防霉处理**：试验时间较长时注意防霉
4. **恢复时间**：试验后应充分恢复再进行检测
