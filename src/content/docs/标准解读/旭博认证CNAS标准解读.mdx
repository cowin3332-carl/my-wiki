## 如何选择适合您的检测标准？

<CardGrid>
  <Card title="1. 确定产品领域" icon="category">
    <p>首先明确您的产品所属的行业范畴，例如军工、轨道交通、汽车电子、民用通用、包装运输或特殊行业应用等。这将帮助您缩小标准选择范围。</p>
  </Card>
  <Card title="2. 明确试验目的" icon="target">
    <p>您的测试是为了产品研发阶段的验证、型式试验（产品鉴定）、还是生产过程中的抽检（环境应力筛选）？不同的目的对应不同的测试严酷等级和标准要求。</p>
  </Card>
  <Card title="3. 查找对应标准" icon="search">
    <p>根据已确定的产品领域和试验目的，在上方分类中查找相应的标准号。例如，车载显示屏通常参考 <a href="/标准解读/各标准解读/gb-t-28046-3-2011">GB/T 28046系列</a>，而地铁上的关键控制器则需符合 <a href="/标准解读/各标准解读/iec-61373">IEC 61373</a> 的要求。</p>
  </Card>
  <Card title="4. 查阅具体参数" icon="settings">
    <p>进入<a href="/标准原文/标准原文">标准原文列表</a>查看具体的标准代码和原始描述，确认是否涵盖了您的产品所需的具体测试条件，如详细的振动量级、温度范围、湿度周期、冲击脉冲等。</p>
  </Card>
</CardGrid>
