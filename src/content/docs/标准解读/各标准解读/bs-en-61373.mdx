## 1. 标准核心参数

<Aside type="tip" title="核心逻辑">
  本标准定义铁路应用电气设备在运行过程中所承受的冲击和振动环境，用于评估设备的抗振能力和结构完整性。
</Aside>

| 参数名称 | 技术指标 | 备注 |
| :--- | :--- | :--- |
| 适用范围 | 铁路车辆电气设备 | 不适用于电子设备抗振 |
| 冲击试验 | 速度变化 0.5-2.0 m/s | 按车辆类型分等级 |
| 振动试验 | 0.7mm (10-50Hz), 1.4mm (0.5-10Hz) | 加速度峰值 5g |
| 轴向 | X, Y, Z 三轴 | 每个轴向 10 次循环 |
| 持续时间 | 15min/轴 | 总计 45min |

---

## 2. 试验流程

<Steps>
1. **预处理**：样品在 25℃±10℃ 存放 24h。
2. **初始检测**：功能检查、外观检查。
3. **冲击试验**：模拟车辆启停、转向等工况。
4. **振动试验**：模拟轨道不平顺引起的振动。
5. **恢复与最后检测**：对比性能参数偏移。
</Steps>

## 4. 实验室能力与夹具建议
- **设备选择**：推荐推力大于 2 倍载荷的电动振动台。
- **夹具要求**：夹具一阶共振频率应高于试验上限频率的 1.5 倍。
- **环境要求**：试验室温度 15-35℃，相对湿度 ≤85%。
